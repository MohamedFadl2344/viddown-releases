## Executive Summary

**VidDown** is a fast, secure, and lightweight desktop application designed to help users manage, process, and download digital media from various platforms. Built using modern technologies (React & Rust), it delivers a premium, ad-free experience with high performance. 

*VidDown is 100% free to use (Freeware), however, its source code is closed-source and not publicly available.*

<br>

## Platform Support
- **Windows:** Officially supported.
- **Linux:** Officially supported.
- **macOS:** Not officially supported yet.

<br>

## Security & Antivirus False Positives
Since VidDown is independently developed and closed-source, it does not yet have a commercial Code Signing Certificate. Some antivirus products rely on Behavior Analysis, which may occasionally lead to false positive warnings for applications packaged with bundlers like PyInstaller (which unpack temporary files at runtime). 
If blocked by Windows SmartScreen, click **"More Info" -> "Run Anyway"**.

<br>

## Legal & Fair Use Policy

VidDown is intended strictly for **legal and fair use**. It allows users to:
- Download and backup their own content  
- Access Public Domain and Creative Commons materials  
- Archive non-copyrighted media  

The application must **not** be used to bypass DRM or violate intellectual property rights. Users are fully responsible for their usage.
